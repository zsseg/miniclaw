# Clawmini 工作区助手 — 帮助手册

## 1. 这是什么

Clawmini 是一款桌面 AI 助手，你只需像聊天一样输入需求，它就能帮你处理文件、写文稿、画图、管理回收站，还能当 QQ 机器人自动回复消息。

> 💡 **联网搜索**：开启后 AI 可检索实时网络信息（DeepSeek/Qwen 支持）。

---

## 2. 启动方式

```powershell
cd 项目目录
python app.py
```

首次启动会自动创建 `workspace/` 工作区目录。看到窗口出现就说明成功了。

---

## 3. 界面布局

应用窗口分三个区域：

| 区域 | 位置 | 用途 |
|------|------|------|
| **会话列表** | 左侧 | 管理多个对话，不同话题分开记录 |
| **功能标签页** | 右侧 | ①文件管理 ②QQ自动回复 |
| **输入框** | 底部 | 在这里和 AI 对话，按 Ctrl+Enter 发送 |
| **齿轮按钮 ⚙** | 右上角 | 打开设置，配置 API Key、模型和**联网搜索** |

> 💡 支持 **DeepSeek** 和 **Qwen** 的联网搜索，在设置中勾选「联网搜索 🔍」即可开启。开启后 AI 在需要时会自动检索互联网信息。

---

## 4. 功能说明

### 4.1 ① 文件管理

这是主操作区。在这里你可以和 AI 对话，让它帮你：
- **文件操作**：创建、编辑、重命名、删除文件
- **图片生成**：描述你想要的画面，AI 用通义万相帮你画出来
- **文稿撰写**：输入主题，AI 自动生成文章，还可以继续修改

页面上方还有快捷的「文稿生成」区域，可以快速填主题生成文章，并预览效果。

### 4.2 ② QQ自动回复

配置 AI 自动回复 QQ 消息。页面内容较多，建议按以下顺序使用：

1. **选网关模式**：测试选 `mock`，正式用 `managed`
2. **下载并安装 NapCat**：点「下载NapCat」按钮，从官网下载一键包安装
3. **点「一键启动/填充」**：自动发现 NapCat 并填充配置
4. **点「更新配置」**：保存群号、延时等设置
5. **先用「模拟收到消息」测试**：确认回复逻辑正常再切换到真实 QQ

> 如果消息里带图片，可以在设置中打开「图片识别」开关，AI 会参考图片内容来回复。

## 6. 联网搜索

> 🔍 支持 **DeepSeek** 和 **Qwen** 模型的联网搜索。

在设置窗口（⚙ 齿轮按钮）中勾选「联网搜索 🔍」，AI 在回答时效性较强的问题（如新闻、天气、最新资讯）时，会自动检索互联网获取实时信息。

开启方式：
1. 点击右上角 ⚙ 齿轮按钮
2. 勾选「联网搜索 🔍（DeepSeek/Qwen 可用）」
3. 点击保存

也可以通过 `/settings enable_search=true` 命令开启。

> ⚠️ 联网搜索需要 DeepSeek 或 Qwen 的 API Key。当前会话的联网状态会在重启后自动恢复。

## 7. 管理命令

在输入框中以 `/` 开头的命令：

### 会话管理

| 命令 | 作用 | 示例 |
|------|------|------|
| `/session new 标题` | 新建会话 | `/session new 学习笔记` |
| `/session list` | 查看所有会话 | `/session list` |
| `/session switch ID` | 切换会话 | `/session switch sess_abc123` |
| `/session rename ID 新标题` | 给会话改名 | `/session rename sess_abc123 归档` |
| `/session delete ID` | 删除会话 | `/session delete sess_abc123` |

### 设置管理

| 命令 | 作用 |
|------|------|
| `/settings` | 查看当前配置 |
| `/settings api_key=sk-xxx` | 设置 API Key |
| `/settings provider=qwen` | 切换模型提供商（qwen / openai / deepseek） |
| `/settings model=qwen-plus` | 切换模型名称 |
| `/settings enable_search=true` | 开启联网搜索 |
| `/settings enable_search=false` | 关闭联网搜索 |

> 也可以点右上角 ⚙ 齿轮按钮，在设置窗口中图形化配置。

## 7. 数据与存储

所有数据保存在 `workspace/` 目录下，重启后自动恢复：

| 文件/目录 | 内容 |

| `sessions/*.json` | 每段对话的历史 |
| `generated_images/` | 生成的图片 |
| `generated_documents/` | 生成的文稿 |
| `app_state.json` | 设置和会话信息 |
| `sessions/*.json` | 每段对话的历史 |
| `generated_images/` | 生成的图片 |
| `generated_documents/` | 生成的文稿 |
| `qq_auto_reply.log` | QQ 机器人的运行日志 |

## 9. 智能意图识别

Clawmini 能自动判断你的输入是**聊天**还是**需要执行任务**：

- **问候/闲聊** → 直接聊天，不调用工具
- **文件操作/图片生成/文稿** → 自动调用对应工具
- **QQ 相关** → 操作 QQ 自动回复设置
- **设置管理** → 查看或修改配置

> 如果你说「帮我画一只猫」，AI 知道这是图片生成任务；如果说「我想学画画」，AI 会和你聊天。这一切都是自动的，无需手动切换模式。

## 10. 安全说明

- 文件操作只限于 `workspace/` 目录，不影响电脑上其他文件
- 删除的文件进回收站，不是永久删除
- QQ 机器人不会修改你的 QQ 客户端

## 11. 常见问题

### Q：启动后窗口空白或闪退？

检查是否在项目根目录运行 `python app.py`，确保 Python ≥ 3.10。

### Q：AI 不响应或报 API 错误？

先设置 API Key。点右上角 ⚙ 齿轮，选择模型提供商（推荐 qwen），填入你的 API Key。

### Q：联网搜索没效果？

确保：① 已勾选「联网搜索 🔍」 ② 使用的模型是 DeepSeek 或 Qwen ③ 问的是有时效性的问题（如「今天有什么新闻」「最近的 AI 进展」）

### Q：图片生成失败？

确认你的 API Key 有通义万相的权限。生成需要 10-30 秒。

### Q：QQ 自动回复不工作？

按照「QQ自动回复」页面的操作指引，先下载 NapCat，点一键启动填充，再更新配置。
也可以先选 `mock` 模式用「模拟收到消息」测试。

### Q：如何换模型？

右上角 ⚙ 设置中更换 provider 和 model，支持 qwen-plus、qwen-max、deepseek-chat、gpt-4o 等。

## 10. 使用技巧

1. **多会话隔离**：不同任务使用不同会话，避免历史混淆
2. **指令越具体越好**：写清楚文件名、路径、格式、风格，减少来回确认
3. **图片生成先测试**：先用小尺寸试跑，确认效果后再正式生成
4. **QQ 先模拟测试**：启动自动回复前先在日志面板观察消息格式
5. **回收站管理**：定期查看回收站，及时清空无用文件释放磁盘空间
